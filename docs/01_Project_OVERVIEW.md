# Project Overview — Karaoke-Style Social Music App (프로젝트 개요)

프로젝트 이름: Karatube Project (초안)
목표(한 문장): 사용자들이 노래(또는 미디어)를 검색하고 예약(큐)에 올려 협업 재생할 수 있는 경량 소셜 뮤직 플랫폼을 만든다.

주요 가치 제안
- 실시간 또는 예약 기반으로 미디어를 재생하고 공유할 수 있다.
- 간단한 큐(Reservation Queue)로 비전문가도 즉시 사용 가능.
- AirPlay 등 외부 출력 지원(우선 iOS MVP).

범위(이 프로젝트의 포함/제외)
- 포함: 검색(YouTube/공개 미디어), 큐 추가/관리, 플레이어(재생/일시정지/다음), 사용자 기본 프로필, 로컬 큐 동기화(간단한 서버/폴링)
- 제외(초기): 결제·광고 플랫폼 통합, 복잡한 권한/ACL, 고가용성 인프라

성공 기준 (MVP)
- 사용자가 검색에서 아이템을 큐에 추가하고, 플레이어에서 정상 재생할 수 있다.
- 협업 큐(한 세션 내 2명 이상)에서 아이템 추가/삭제가 반영된다(간단한 동기화).
- 기본적인 보안정책(토큰 관리·노출 금지)이 문서화되어 있다.
